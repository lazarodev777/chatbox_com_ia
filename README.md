# chatbox_com_ia
# 🤖 Chatbot com IA usando Streamlit e Gemini

Este projeto é um **chatbot interativo desenvolvido em Python** utilizando **Streamlit** para a interface web e a **API do Google Gemini** para geração de respostas com Inteligência Artificial.

A aplicação permite que o usuário converse com a IA em tempo real através de um chat simples e intuitivo diretamente no navegador.

## 🚀 Funcionalidades

* Interface de chat interativa
* Integração com a API do **Google Gemini**
* Histórico de mensagens durante a sessão
* Campo para o usuário informar seu nome
* Feedback visual enquanto a IA gera a resposta
* Estrutura simples e organizada para aprendizado de IA com Python

## 🛠️ Tecnologias utilizadas

* **Python**
* **Streamlit**
* **Google Generative AI (Gemini)**
* **python-dotenv**
* **Git / GitHub**

## ⚙️ Como executar o projeto

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/chatbox_com_ia.git
```

2. Instale as dependências

```bash
pip install -r requirements.txt
```

3. Crie um arquivo `.env` e adicione sua chave da API:

```
GOOGLE_API_KEY=sua_chave_aqui
```

4. Execute o projeto

```bash
streamlit run chat.py
```

## 💡 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de **praticar integração de APIs de Inteligência Artificial com aplicações web em Python**, além de demonstrar o uso do **Streamlit para criação rápida de interfaces interativas**.
